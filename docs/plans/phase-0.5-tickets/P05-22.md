## Context

P05-17 created the test execution engine: `runTest(script, resolver, options) → TestResult`. This ticket wraps it in a panel that shows:
- Run button to execute the current chip's `.tst` against the student's implementation
- Progress: "Running step 3 of 8"
- Output table: columns from `output-list`, rows from test execution
- Diff highlighting: cells where actual ≠ expected shown in red
- Summary: "Comparison ended successfully" or "Comparison failure at row N, column 'out'"

### Integration points:

- **P05-19 (curriculum browser)**: When a chip is selected, its `.tst` and `.cmp` are available via `getChipFiles(name)`.
- **P05-17 (engine)**: `runTest()` returns `TestResult` with `outputRows`, `firstFailure`, `passed`.
- **P05-09 (StatusBar)**: Post pass/fail message after test completes.
- **P05-19 completion tracking**: When test passes, mark chip as completed in localStorage.

---

## Goal

After this ticket:
1. `TestResultsPanel` shows a "Run Test" button
2. Clicking Run executes the current chip's `.tst` via `runTest()` (optionally shows **progress** text: step N of M — derive from `outputRows.length` while running or from script command count)
3. Output table renders with column headers from `output-list`
4. Passing rows: normal. Failing row: red highlight on mismatched cell
5. Summary line: "Comparison ended successfully" (green) or failure detail (red)
6. Pass result → marks chip as completed in P05-19's localStorage tracker

### `src/components/ui/TestResultsPanel.tsx`

```tsx
import { useState } from 'react'
import { Button, Typography } from 'antd'
import { runTest } from '@/core/testing/nand2tetris/engine'
import { parseTST } from '@/core/testing/nand2tetris/tstParser'
import { parseCmp } from '@/core/testing/nand2tetris/cmpParser'
import { getChipFiles } from '@/data/nand2tetris/project01'
import type { TestResult } from '@/core/testing/nand2tetris/engine'

const { Text } = Typography

interface Props {
  chipName: string | null
  chipResolver: import('@/core/testing/nand2tetris/engine').ChipResolver | null
}

export function TestResultsPanel({ chipName, chipResolver }: Props) {
  const [result, setResult] = useState<TestResult | null>(null)
  const [running, setRunning] = useState(false)
  const [columns, setColumns] = useState<string[]>([])

  const handleRun = () => {
    if (!chipName || !chipResolver) return

    const files = getChipFiles(chipName)

    setRunning(true)

    const tstParsed = parseTST(files.tst)
    if (!tstParsed.success) {
      setResult({ passed: false, totalSteps: 0, passedSteps: 0, outputRows: [], firstFailure: null, error: `TST parse error: ${tstParsed.errors[0]?.message}` })
      setRunning(false)
      return
    }

    const cmpData = parseCmp(files.cmp)
    const testResult = runTest(tstParsed.script, chipResolver, {
      cmpData,
      // If .tst uses `compare-to Other.cmp`, implement loader mapping filename → bundled cmp (P05-07); see P05-17.
      loadCmpFile: (fileName) => {
        const name = fileName.replace(/\.cmp$/i, '')
        try {
          return parseCmp(getChipFiles(name).cmp)
        } catch {
          return null
        }
      },
    })

    // Extract columns from output-list
    const outputListCmd = tstParsed.script.commands.find(c => c.type === 'output-list')
    if (outputListCmd && outputListCmd.type === 'output-list') {
      setColumns(outputListCmd.columns.map(c => c.name))
    }

    setResult(testResult)
    setRunning(false)

    // Completion tracking — **same format as P05-19** (`JSON.stringify([...Set])` or Set round-trip; do not use one array here and Set there).
    if (testResult.passed) {
      try {
        const saved = localStorage.getItem('hacer-completed-chips')
        const completed: string[] = saved ? [...new Set(JSON.parse(saved) as string[])] : []
        if (!completed.includes(chipName)) {
          completed.push(chipName)
          localStorage.setItem('hacer-completed-chips', JSON.stringify(completed))
        }
      } catch { /* ignore */ }
    }
  }

  return (
    <div data-testid="test-results-panel" style={{ padding: 8 }}>
      <Button
        data-testid="run-test-button"
        type="primary"
        size="small"
        onClick={handleRun}
        loading={running}
        disabled={!chipName || !chipResolver}
      >
        Run Test
      </Button>

      {result && (
        <>
          <div data-testid="test-summary" style={{ marginTop: 8, fontSize: 12 }}>
            {result.error ? (
              <Text type="danger">{result.error}</Text>
            ) : result.passed ? (
              <Text style={{ color: '#52c41a' }}>Comparison ended successfully</Text>
            ) : (
              <Text type="danger">
                Comparison failure at row {result.firstFailure?.row}, column &apos;{result.firstFailure?.column}&apos;:
                expected {result.firstFailure?.expected}, got {result.firstFailure?.actual}
              </Text>
            )}
          </div>

          {result.outputRows.length > 0 && columns.length > 0 && (
            <div data-testid="output-table" style={{ marginTop: 8, overflowX: 'auto' }}>
              <table style={{ borderCollapse: 'collapse', fontSize: 11, fontFamily: 'monospace' }}>
                <thead>
                  <tr>
                    <th style={{ padding: '2px 6px', borderBottom: '1px solid #444' }}>#</th>
                    {columns.map(col => (
                      <th key={col} style={{ padding: '2px 6px', borderBottom: '1px solid #444' }}>{col}</th>
                    ))}
                  </tr>
                </thead>
                <tbody>
                  {result.outputRows.map((row, i) => {
                    const isFailing = !result.passed && result.firstFailure?.row === i
                    return (
                      <tr key={i} data-testid={`output-row-${i}`}>
                        <td style={{ padding: '2px 6px', color: '#888' }}>{i}</td>
                        {columns.map(col => {
                          const isFailCol = isFailing && result.firstFailure?.column === col
                          return (
                            <td
                              key={col}
                              data-testid={isFailCol ? 'fail-cell' : undefined}
                              style={{
                                padding: '2px 6px',
                                background: isFailCol ? '#4a1111' : 'transparent',
                                color: isFailCol ? '#ff4d4f' : '#e0e0e0',
                              }}
                            >
                              {row.values[col] ?? '-'}
                            </td>
                          )
                        })}
                      </tr>
                    )
                  })}
                </tbody>
              </table>
            </div>
          )}

          <div style={{ marginTop: 4, fontSize: 11, color: '#888' }}>
            Steps: {result.passedSteps}/{result.totalSteps}
          </div>
        </>
      )}
    </div>
  )
}
```

### `src/components/ui/TestResultsPanel.test.tsx`

```typescript
import { describe, it, expect, beforeEach, vi } from 'vitest'
import { render, screen, fireEvent } from '@testing-library/react'
import { TestResultsPanel } from './TestResultsPanel'
import type { ChipResolver, ChipEvaluator } from '@/core/testing/nand2tetris/engine'

function makeNotResolver(): ChipResolver {
  return {
    resolve(chipName) {
      if (chipName !== 'Not') return null
      const inputs: Record<string, number> = {}
      const outputs: Record<string, number> = {}
      return {
        setInput(name, value) { inputs[name] = value },
        evaluate() { outputs.out = inputs.in === 0 ? 1 : 0 },
        getOutput(name) { return outputs[name] },
        getInput(name) { return inputs[name] ?? 0 },
      }
    },
  }
}

function makeBrokenResolver(): ChipResolver {
  return {
    resolve() {
      const inputs: Record<string, number> = {}
      return {
        setInput(name, value) { inputs[name] = value },
        evaluate() {},
        getOutput() { return 0 },
        getInput(name) { return inputs[name] ?? 0 },
      }
    },
  }
}

beforeEach(() => {
  localStorage.clear()
})

describe('TestResultsPanel', () => {
  it('renders Run Test button', () => {
    render(<TestResultsPanel chipName="Not" chipResolver={makeNotResolver()} />)
    expect(screen.getByTestId('run-test-button')).toBeTruthy()
  })

  it('Run Test button is disabled when no chip selected', () => {
    render(<TestResultsPanel chipName={null} chipResolver={null} />)
    expect(screen.getByTestId('run-test-button').hasAttribute('disabled') ||
           screen.getByTestId('run-test-button').getAttribute('aria-disabled') === 'true').toBe(true)
  })

  it('running Not test shows success summary', () => {
    render(<TestResultsPanel chipName="Not" chipResolver={makeNotResolver()} />)
    fireEvent.click(screen.getByTestId('run-test-button'))
    expect(screen.getByTestId('test-summary').textContent).toContain('Comparison ended successfully')
  })

  it('running broken chip shows failure summary', () => {
    render(<TestResultsPanel chipName="Not" chipResolver={makeBrokenResolver()} />)
    fireEvent.click(screen.getByTestId('run-test-button'))
    const summary = screen.getByTestId('test-summary')
    expect(summary.textContent).toContain('Comparison failure')
    expect(summary.textContent).toContain('out')
  })

  it('output table renders rows', () => {
    render(<TestResultsPanel chipName="Not" chipResolver={makeNotResolver()} />)
    fireEvent.click(screen.getByTestId('run-test-button'))
    expect(screen.getByTestId('output-table')).toBeTruthy()
    expect(screen.getByTestId('output-row-0')).toBeTruthy()
  })

  it('failing cell is highlighted', () => {
    render(<TestResultsPanel chipName="Not" chipResolver={makeBrokenResolver()} />)
    fireEvent.click(screen.getByTestId('run-test-button'))
    expect(screen.getByTestId('fail-cell')).toBeTruthy()
  })

  it('passing test marks chip as completed in localStorage', () => {
    render(<TestResultsPanel chipName="Not" chipResolver={makeNotResolver()} />)
    fireEvent.click(screen.getByTestId('run-test-button'))
    const saved = JSON.parse(localStorage.getItem('hacer-completed-chips') ?? '[]')
    expect(saved).toContain('Not')
  })
})
```

## Pitfalls

- **`getChipFiles` dependency (P05-07)**: If curriculum data not yet merged, tests can inline small `.tst`/`.cmp` strings for Not. Add the `getChipFiles` integration later.
- **Synchronous vs async**: `runTest()` is synchronous (all 16 Project 1 tests run in <100ms). No need for async/workers for Layer 3. If perf becomes an issue in later projects, move to a web worker.
- **Antd `Table` vs plain `<table>`**: Antd `Table` is heavy for small data. A plain `<table>` with inline styles is simpler and faster for the test output (typically 2–16 rows).
- **`firstFailure` only**: Currently the engine reports only the **first** failure. For Layer 3 this is sufficient. A future enhancement could report all mismatches.
- **Completion tracking race**: Multiple rapid runs could cause concurrent localStorage writes. The simple read-append-write pattern is fine for single-tab usage.
- **localStorage schema**: P05-19 must persist completion as **`JSON.stringify(string[])`** (not a raw `Set`). If CurriculumBrowser still uses `new Set(JSON.parse(...))` in memory, keep the **same key** `hacer-completed-chips` and array-on-wire format.
- **`loadCmpFile` mapping**: The sketch maps `Foo.cmp` → `getChipFiles('Foo')` — valid when the basename matches the chip module name; if not, maintain an explicit map.
- **UI thread**: `runTest` is synchronous; for a visible progress indicator, either chunk work with `queueMicrotask`/`setTimeout(0)` between steps (optional) or show an indeterminate spinner while `running === true`.
- **React Compiler**: No `useMemo`, `useCallback`, `React.memo`.
