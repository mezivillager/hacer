# P05-09 StatusBar Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a bottom status bar backed by Zustand actions so HACER can surface info, warning, and error feedback from parsing, validation, and test execution flows.

**Architecture:** Add a focused status slice to the existing circuit store (`statusMessages` state + `addStatus`/`clearStatus`/`clearAllStatus` actions), then render only the latest message in a dedicated UI component. Keep severity handling explicit in both state and UI, and expose the actions through the existing `circuitActions` facade so future tickets and Playwright store tests can call them consistently.

**Tech Stack:** React 19, TypeScript strict, Zustand + Immer, Ant Design layout, Vitest, Playwright store tests.

```ts
// src/store/actions/statusActions/statusActions.ts
import type { CircuitStore, StatusActions, StatusMessage } from '../../types'

type SetState = (
  fn: (state: CircuitStore) => void,
  replace?: false,
  actionName?: string
) => void

let statusCounter = 0

// Note: ID values are monotonic across test runs; avoid exact-ID assertions.

export const createStatusActions = (set: SetState): StatusActions => ({
  addStatus: (severity, text) => {
    const message: StatusMessage = {
      id: `status-${++statusCounter}`,
      severity,
      text,
      timestamp: Date.now(),
    }

    set((state) => {
      state.statusMessages.push(message)
    }, false, 'addStatus')

    return message
  },

  clearStatus: (id) => {
    set((state) => {
      state.statusMessages = state.statusMessages.filter((msg) => msg.id !== id)
    }, false, 'clearStatus')
  },

  clearAllStatus: () => {
    set((state) => {
      state.statusMessages = []
    }, false, 'clearAllStatus')
  },
})
```

```tsx
import { useCircuitStore, circuitActions } from '@/store/circuitStore'

const SEVERITY_COLORS = {
  info: '#1677ff',
  warning: '#faad14',
  error: '#ff4d4f',
} as const

export function StatusBar() {
  const messages = useCircuitStore((state) => state.statusMessages)
  const latest = messages[messages.length - 1]

  if (!latest) return null

  return (
    <div
      data-testid="status-bar"
      data-severity={latest.severity}
      role="status"
      aria-live="polite"
      onClick={() => circuitActions.clearStatus(latest.id)}
      style={{
        position: 'fixed',
        left: 0,
        right: 0,
        bottom: 0,
        zIndex: 1000,
        display: 'flex',
        justifyContent: 'space-between',
        alignItems: 'center',
        padding: '6px 12px',
        color: '#ffffff',
        cursor: 'pointer',
        backgroundColor: SEVERITY_COLORS[latest.severity],
      }}
    >
      <span data-testid="status-text">{latest.text}</span>
      <span aria-hidden style={{ opacity: 0.8 }}>click to dismiss</span>
    </div>
  )
}
```
