# Tasks

This folder supports workflow orchestration for AI agents and developers. See [docs/llm-workflow.md](../docs/llm-workflow.md) for the full workflow guide.

## Files

| File | Purpose |
|------|---------|
| **todo.md** | Current task plan with checkable items. Plan first, track progress, document results. |
| **lessons.md** | Captured patterns and mistakes. Update after any correction; review at session start. |

## When to Use

- **todo.md**: Create/update when starting a non-trivial task (3+ steps). Use checkable items (`- [ ]`). Add a review section on completion.
- **lessons.md**: Update after *any* user correction. Document the pattern or mistake and a rule to prevent recurrence.

## Templates

- [todo.md.template](./todo.md.template) — Copy structure for new task plans
- [lessons.md.template](./lessons.md.template) — Copy structure for capturing lessons
